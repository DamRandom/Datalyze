# Datalyze Frontend

This repository contains the frontend code for **Datalyze**, a predictive system designed for analyzing and visualizing the teaching staff composition at the University of Computer Sciences (UCI). Built with **Next.js** and **Tailwind CSS**, it provides a professional interface for interacting with predictive models and supporting decision-making.

## Features

- Interactive data visualization.
- Integration with predictive models.
- Optimized performance with Next.js.
- Fully responsive UI with Tailwind CSS.

## Installation

1. Clone the repository:

   ```sh
   git clone https://github.com/yourusername/Datalyze.git
   ```

2. Navigate to the project directory:

   ```sh
   cd Datalyze
   ```

3. Install dependencies:

   ```sh
   npm install
   ```

4. Start the development server:

   ```sh
   npm run dev
   ```
